# AI & ML Internship - Task 3: Linear Regression

## Objective
Implement and understand simple & multiple linear regression using Python.

## Tools
- Python
- Pandas
- NumPy
- Matplotlib & Seaborn
- scikit-learn

## Dataset
We used a sample House Price dataset with features:
- `Size_sqft`
- `Num_Rooms`
- `Age_Years`
- `Price` (target)

## Steps Performed
1. Imported libraries and created sample dataset.
2. Preprocessed the dataset (split features & target).
3. Split data into train and test sets.
4. Fit a Linear Regression model using `sklearn.linear_model`.
5. Evaluated model using MAE, MSE, and R² score.
6. Interpreted model coefficients.
7. Visualized regression line and correlation heatmap.
8. Saved plots in `outputs/figures/`.
